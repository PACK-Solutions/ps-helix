Bonnes Pratiques

Utilisation HTML

<i class="ph ph-house" aria-hidden="true"></i>
  • Toujours utiliser aria-hidden="true" pour les icônes décoratives
  • Respecter la nomenclature ph-[nom-icône]
  • Préférer les icônes sémantiques

Accessibilité

<button aria-label="Paramètres">
  <i class="ph ph-gear" aria-hidden="true"></i>
</button>
  • Fournir un aria-label pour les éléments interactifs
  • Masquer les icônes décoratives des lecteurs d'écran
  • Assurer un contraste suffisant

Personnalisation

.ph {
  font-size: 1.5rem;
  color: var(--primary-color);
  transition: color 0.2s ease;
}
  • Utiliser les variables CSS pour la cohérence
  • Maintenir des tailles cohérentes
  • Ajouter des transitions fluides
