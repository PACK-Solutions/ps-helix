Auto-dimensionnement

Activez [autoSize]="true" : un effect() ajuste la hauteur du textarea à son contenu.

Documentation API

Le composant psh-textarea est hybride : il implémente ControlValueAccessor pour les Reactive Forms et expose un model<string> compatible FormValueControl (Signal Forms, Angular 21+). Un seul signal value sert de source de vérité aux deux contrats.

Propriétés (Model Inputs)

Propriétés à liaison bidirectionnelle ([(ngModel)], [(value)], [formControl]).

Propriété Type Défaut Description
value string '' Valeur courante du textarea (source de vérité partagée CVA / Signal Forms)
disabled boolean false Désactive le champ et empêche toute interaction
readonly boolean false Champ visible mais non modifiable

Propriétés (Input Signals)

Propriétés de configuration unidirectionnelles.

Propriété Type Défaut Description
label string requis Texte du label associé (déclaré via input.required())
placeholder string '' Texte indicatif affiché quand le champ est vide
helperText string '' Texte d'aide affiché sous le champ
rows number 4 Nombre de lignes visibles par défaut
maxLength number | undefined undefined Longueur maximale autorisée (déclenche un état erreur si dépassée)
showCharacterCount boolean false Affiche le compteur de caractères (aria-live="polite")
resize TextareaResize 'vertical' Sens de redimensionnement (none | vertical | horizontal | both)
autoSize boolean false Ajuste la hauteur au contenu via effect()
variant TextareaVariant 'outlined' Style visuel (outlined | filled)
size TextareaSize 'medium' Taille du champ (small | medium | large)
fullWidth boolean false Supprime la largeur maximale par défaut
required boolean false Marque le champ comme requis (aria-required + astérisque)
showLabel boolean true Affiche ou masque visuellement le label
ariaLabel string | null null Label ARIA personnalisé (prioritaire sur label)
error string | null null Message d'erreur explicite (force l'état erreur)

Événements (Outputs)

Événement Type Description
valueChange EventEmitter<string> Émis à chaque modification de la valeur
textareaFocus EventEmitter<void> Émis lorsque le champ reçoit le focus
textareaBlur EventEmitter<void> Émis lorsque le champ perd le focus (déclenche aussi onTouched)

Intégration Forms

Le composant détecte automatiquement les deux API de formulaires Angular.

API Contrat Liaison template Détection d'erreur
Reactive Forms ControlValueAccessor [formControl], [formControlName], [(ngModel)] Via NgControl injecté : invalid && (touched || dirty)
Signal Forms (Angular 21+) FormValueControl<string> [field]="form.xxx" Propagée par la directive FormField
Signal two-way (Angular 20) model<string> [(value)]="mySignal" Via l'input error ou dépassement de maxLength
